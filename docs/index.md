# Nexacon Live Chat Widget

Embeddable live chat widget for any website. Add real-time support to your site with a single `<script>` tag.

## Install in 10 seconds

```html
<script
  src="https://unpkg.com/nexacon-livechat-widget/dist/nexacon-chat.min.js"
  data-widget-id="YOUR_WIDGET_UUID"
></script>
```

That's it. A chat bubble appears in the bottom-right corner. Visitors can chat instantly — no login required.

## What you need

- A **Widget ID** from your Nexacon dashboard (created by your developer/admin)
- Nothing else — no API keys, no server setup on the website

## Features

- 💬 Real-time chat via XMPP WebSocket
- 🤖 Routes to AI bot or live agent automatically
- 🎨 Customizable colors and welcome message
- 📱 Mobile responsive
- ⚡ 14kb minified, zero runtime dependencies
