Nexacon Live Chat Widget Demo

This page embeds the live chat widget. The chat bubble should appear in the bottom-right corner.

Method 1: Script tag (auto-init)

<script
  src="nexacon-chat.js"
  data-widget-id="YOUR_WIDGET_UUID"
  data-visitor-name="John Doe"
  data-visitor-email="john@example.com"
></script>

Method 2: Programmatic init

<script src="nexacon-chat.js"></script>
<script>
  NexaconChat.init({
    widgetId: 'YOUR_WIDGET_UUID',
    visitorName: 'John Doe',
    visitorEmail: 'john@example.com',
    baseUrl: 'https://your-nexacon-server.com/api/v1.0',
    ejabberdWsUrl: 'wss://your-ejabberd-server.com:5443/ws',
  });
</script>

The widget only needs a widget ID. API key and secret key are kept on the server — never exposed to visitors.
